# YouTube Watch Later Randomizer, Exporter &amp; Tracker

A user script to randomly open a random YouTube video from your Watch-later playlist.
- I made this to empty my Watch-later playlist in a fun way
Usage:
- Install Tampermonkey add-on
- Install the script
- go to the Watch-later playlist
- A red 🎲 button should appear on the bottom right of the page

IMPORTANT:
to export hidden "Unavailable Videos" you have to show them first (press the three dots on the left and chose Show unavailable videos).
